Preguntas Frecuentes

¿Es gratis el simulador de redes?

Sí, puedes usar el simulador online sin instalación. No requiere registro ni descarga de software adicional. Funciona directamente desde cualquier navegador moderno.

¿Sirve para practicar CCNA?

Sí, el simulador permite practicar routing, switching, VLAN, configuración de topologías reales y troubleshooting. Es una excelente herramienta complementaria para la certificación Cisco CCNA.

¿Es alternativa a Packet Tracer?

Sí, funciona directamente desde el navegador sin necesidad de instalar ningún programa. Es una alternativa web a Cisco Packet Tracer, GNS3 y EVE-NG para prácticas de redes.

¿Puedo simular redes FTTH/GPON?

Sí, el simulador incluye equipos OLT y ONT para diseñar y probar redes de fibra óptica FTTH con tecnología GPON, incluyendo splitters y medición de potencias ópticas.

¿Qué tipo de equipos puedo simular?

Puedes trabajar con routers, switches (incluyendo PoE), firewalls, SDWAN, access points WiFi, balanceadores de carga, cámaras IP, servidores, terminales de pago, impresoras y equipos especializados GPON.
